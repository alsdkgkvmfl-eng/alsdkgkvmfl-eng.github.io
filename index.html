C언어를 배울 때 가장 먼저 작성하는 기본 코드 구조를 분석했습니다. printf 함수를 사용하기 위해 헤더 파일을 포함하는 것이 핵심입니다. 
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

변수와 자료형 이해하기

작성일: 2026년 6월

정수를 저장하는 int, 실수를 저장하는 float/double, 문자를 저장하는 char 자료형의 크기와 선언 방법을 정리했습니다. 
#include <stdio.h>

int main() {
    int age = 20;
    float height = 175.5;
    char grade = 'A';
    
    printf("나이: %d, 키: %.1f\n", age, height);
    return 0;
}

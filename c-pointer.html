포인터와 메모리 구조

상위 항목: 컴퓨터 공학 학습 아카이브

1. 개요

C언어에서 포인터(Pointer)는 메모리의 주소값을 직접 가리키는 변수 유형입니다. 이 특성은 대량의 데이터를 효율적으로 다루거나 소켓 프로그래밍에서 통신 버퍼를 핸들링할 때 핵심적인 역할을 합니다.

2. 기본 예제

포인터 변수를 선언할 때는 변수명 앞에 애스터리스크(*)를 붙여 선언합니다.

int target = 42;
int *ptr = &target; // target의 메모리 주소를 ptr에 저장

여기서 가리키는 주소값을 따라가 내부 데이터를 위조하거나 분석하는 기술은 시스템 해킹이나 패킷 & 리버스 엔지니어링의 기초 학문이 됩니다.
